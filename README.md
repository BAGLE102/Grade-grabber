# Mooodle成績整理工具 

  學校的Moodle教學平台在管理學生成績時效率低下，必須手動將成績匯出至
Excel，這一過程不僅耗時，且容易出現人工錯誤。此軟體通過模擬使用者登入
Moodle平台，自動抓取學生成績資料，並將其匯出為Excel檔案。通過這一自動
化解決方案，減少了人工操作的時間與錯誤率。
